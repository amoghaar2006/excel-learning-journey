# Cell Referencing in Excel

## 1. Relative Referencing
- Used for CGPA calculation and row-wise addition
- Formula example: =C20 + D20
- Changes automatically when dragged

## 2. Absolute Referencing
- Used for fixed discount value
- Formula: =B2 * (1 - $C$1)
- $ used to lock cell reference

## 3. Mixed Referencing
- Used for multi-directional calculations (Day 1, Day 2)
- Formula: =$E32 * (1 - I$32)
- Fixes either row or column
