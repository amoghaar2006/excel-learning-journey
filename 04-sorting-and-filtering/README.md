# Sorting and Filtering in Excel

## Overview
This section demonstrates how data can be organized and arranged using Excel's sorting functionality. The goal is to understand how to sort data based on single and multiple criteria.

---

## Tasks Performed

### 1. Sorting Based on Percentage
- Calculated total marks using relative referencing
- Calculated percentage using cell referencing
- Sorted the data based on percentage (Largest to Smallest)

### 2. Multi-Level Sorting
- Applied additional sorting levels to handle tie cases
- Sorting priority:
  1. Percentage (Primary)
  2. Physics marks
  3. Chemistry marks
  4. Mathematics marks

---

### 3. Alphabetical Sorting
- Created a dataset of names
- Sorted the names in ascending order (A to Z)

---

## Key Learnings
- Understanding how sorting affects data arrangement
- Using single-level and multi-level sorting
- Organizing data for better readability and analysis

---

## Notes
- Sorting was performed using the Data → Sort option in Excel
- Focus is on understanding sorting logic rather than interface steps
