# Basic Functions in MS Excel

## Overview
This section demonstrates fundamental Excel operations used for performing basic calculations. The goal is to understand how Excel handles arithmetic operations, cell referencing, and built-in functions.

## Objectives
- Understand different ways to perform addition
- Learn how to use cell references instead of manual calculations
- Explore commonly used Excel functions
- Get familiar with structured references in tables

---

## Concepts Covered

### 1. Direct Addition
Performed manual addition using values directly.

Example:
23 + 34 = 57

---

### 2. Cell Referencing
Used cell references to perform addition instead of typing values manually.

Example:
= A4 + B4

This allows dynamic calculation when values change.

---

### 3. SUM Function
Used the built-in SUM function to calculate the total of multiple values.

Example:
= SUM(A24:A32)

This is more efficient than adding values one by one.

---

### 4. Exponentiation (Square of a Number)
Calculated the square of numbers using the exponent operator.

Example:
= C24^2

AutoFill was used to apply the formula to multiple rows.

---

### 5. Square Root Function
Calculated the square root of a number using the SQRT function.

Example:
= SQRT(G24)

---

### 6. Table-Based Referencing (Structured Referencing)
Performed calculations using structured references in a table.

Example:
= [@Column1] + [@Column2]

This approach improves readability and is useful in structured datasets.

---

## Key Learnings
- Difference between manual calculations and formula-based calculations
- Importance of cell referencing in dynamic data handling
- Usage of built-in Excel functions for efficiency
- Introduction to structured references in tables
