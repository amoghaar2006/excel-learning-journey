# Student Marks Analysis (Basic)

## Overview
This project involves converting raw student data into a structured table and performing basic analysis using Excel.

## Steps Performed
- Converted raw data into a formatted table
- Calculated total marks for each student using relative referencing
- Calculated percentage based on total marks
- Applied basic styling and formatting for better readability

## Formula Used
- Total Marks:
  = SUM(Physics, Chemistry, Mathematics)

- Percentage:
  = Total / 300 * 100

## Notes
- Performance-based colors (BEST, GOOD, AVERAGE, BELOW AVERAGE) are applied manually based on percentage ranges
- Conditional formatting and logic-based automation (e.g., IF formulas) are not used yet and will be implemented in future updates

## Concepts Used
- Relative Referencing
- AutoFill
- Basic arithmetic operations
- Table formatting and styling
